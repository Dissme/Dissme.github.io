## MicroComponent

> 微前端组件

这个组件用来渲染远程的组件 配合 [onLoadMicro](#onLoadMicro) 使用

```jsx
import { MicroComponent } from '@easythings/easy-view/jsx-runtime';
const url = '/main.worker.jsx';
function Main() {
  return <MicroComponent url={url} />;
}
```
