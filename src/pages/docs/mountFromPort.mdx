## mountFromPort(port: MessagePort, container: HTMLElement): () => void

> 把从 port 收到的消息渲染到页面上

**port:** 一个可以收消息的 MessagePort\
其他和 [mount](#mount) 一样

```jsx
const port = new Worker(new URL('@/components/worker/Menu', import.meta.url));
const destroy = mountFromPort(port, document.querySelector('#menu')); // 挂载
destroy(); // 卸载
```
